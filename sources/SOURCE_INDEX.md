# Source Index

This directory is intentionally minimal.

The public claim for this repo is:

`Evidence Gate: The Reliability Layer for AI Agents.`

This repo now stands on its own through its implementation, API surface,
benchmark proof, and partner-facing docs. It does not need to broadcast the
exploratory trading or Quantum Field Harmonics material that preceded it.

## Public Source Policy

Keep only source material that directly supports review or use of this product:

- current product and execution docs
- benchmark cases and results
- implementation-facing notes that explain the shipped system

## Archive Policy

The public repo intentionally does not ship the exploratory whitepapers, raw
PDFs, extracted text files, trading notes, or QFH research archive.

If that historical material needs to be retained for provenance, keep it in a
private archive or separate private storage rather than on the public GitHub
surface.

## Working Rule

If a source artifact does not help a prospective reviewer understand, run, or
evaluate Evidence Gate directly, it does not belong in this public repository.
